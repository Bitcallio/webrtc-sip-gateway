# @bitcall/webrtc-sip-gateway

CLI package that bootstraps and manages the Bitcall WebRTC-SIP Gateway deployment on Linux hosts.

## Commands

- `sudo bitcall-gateway init`
- `sudo bitcall-gateway up`
- `sudo bitcall-gateway down`
- `sudo bitcall-gateway status`
- `sudo bitcall-gateway logs [service]`
